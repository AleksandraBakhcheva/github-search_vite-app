# Search for repositories 

App based on GitHub API allows you to search for repositories based on input and shows details for every found project

## Stack

- React
- MobX
- JavaScript
- React Router
- Sass
- Framer Motion
- Vite
  
## Start app

```bash
clone the project
cd "target folder"
npm install
npm run dev
```
